Project ↗     Git Repo ↗

Development Bootcamp 3

In the next crash course, we learnt about CSS animations. This is quite a new concept to me as I did not really attempt them last semester, however I am really interested in learning how to do them as I would like to implement them into my portfolio. For this weeks challenge, we were tasked with recreating something from 'CSS Battles' and adding animations to it.

I really enjoyed the first part of this challenge while trying to recreate the element given, and this part went well. However, I ran into some issues when trying to implement the animations.

Project ↗

"Great job with the animations and transitions—they're well done, and I love that you used unique transitions for each property! To improve, consider making the elements siblings instead of children to fix the growing overlap issue. For styling, using one class (e.g., .rectangle) with :nth-child(x) could simplify the CSS for similar rectangles. Also, using flexbox could help ensure the transition is triggered only on the cross area, not the larger block space."

- Maikel
